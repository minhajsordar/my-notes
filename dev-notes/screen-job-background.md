## install screen 
sudo apt update
sudo apt install screen

## create new screen 
screen -S screenname

## detach screen
Ctrl+A Ctrl+D

### detach screen from anywhere eles
screen -S  screen name and session

## list of screens 
screen -ls

## atach screen 
screen -r screen name and session

## kill screen 
reatach and then type "exit" + enter to exit
